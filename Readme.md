# 🏦 Loan Approval Prediction API

A Machine Learning-powered REST API that predicts whether a loan application is likely to be **approved or rejected** based on an applicant's demographic and financial information.

The project was developed using **Python**, **Scikit-learn**, and **FastAPI**, containerized with **Docker**, and deployed on **AWS EC2**.

---

## 📖 Overview

Loan approval is a critical process in the banking and financial sector. This project automates that process by training multiple machine learning models on historical loan application data and exposing the best-performing model through a production-ready REST API.

Users can submit applicant details through the API and receive an instant prediction.

# 📊 Model Performance

The following machine learning algorithms were evaluated.

| Model | Accuracy |
|--------|----------|
| Random Forest | 78.57% |
| Gaussian Naive Bayes | **83.12%** |
| Decision Tree | 72.08% |
| K-Nearest Neighbors | 81.17% |
| Support Vector Classifier | 83.12% |

The deployed API uses the **Gaussian Naive Bayes** classifier, which achieved the highest validation accuracy.

# 📋 Input Features

The model expects the following applicant information.

| Feature | Description |
|----------|-------------|
| Gender | Applicant Gender |
| Married | Marital Status |
| Dependents | Number of Dependents |
| Education | Education Level |
| Self Employed | Employment Status |
| Applicant Income | Monthly Applicant Income |
| Coapplicant Income | Monthly Co-applicant Income |
| Loan Amount | Requested Loan Amount |
| Loan Amount Term | Loan Repayment Term |
| Credit History | Credit Score Availability |
| Property Area | Urban / Semiurban / Rural |

# 🚀 API Endpoints

## Home

```
GET /
```

Returns a welcome message.

---

## Predict Loan Approval

```
POST /predict
```

### Example Request

```json
{
    "gender": "Male",
    "married": "Yes",
    "dependents": 1,
    "education": "Graduate",
    "self_employed": "No",
    "applicant_income": 5000,
    "coapplicant_income": 2000,
    "loan_amount": 150,
    "loan_amount_term": 360,
    "credit_history": 1,
    "property_area": "Urban"
}
```

### Example Response

```json
{
    "loan_status": "Approved"
}
```

# ⚙️ Installation

Clone the repository.

```bash
git clone https://github.com/yourusername/loan-approval-api.git
cd loan-approval-api
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Start the FastAPI server.

```bash
uvicorn main:app --reload
```

Open

```
http://127.0.0.1:8000/docs
```

---

# 🐳 Docker

Build the Docker image.

```bash
docker build -t loan-api .
```

Run the container.

```bash
docker run -p 8000:8000 loan-api
```
